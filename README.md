# Cross Domain target recognition for SAR images
## A method for cross domain target recognition only using source domain data
Although there are some domain adaptation methods to achieve cross domain object recognition, they still require access to unlabeled data in the target domain.
From the existing methods, they rely on enhancing the generation of data. But is generating additional enhanced data really just for obtaining cross domain universal features? What impact did these enhanced data actually have on the training of the model?
